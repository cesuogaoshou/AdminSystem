# AdminSystem

企业后台管理系统练习项目，目标是完整实现一个标准 Java 后端后台系统，覆盖：

- 用户、角色、菜单、部门管理
- RBAC 权限模型
- JWT 登录认证
- 操作日志审计
- 数据字典缓存
- MySQL、Redis、RabbitMQ、Docker Compose
- 单元测试和基础集成测试

## 技术栈

- Java 21
- Spring Boot 3.5.x
- Maven 3.9.x
- MyBatis
- MySQL 8.4
- Redis 7.x
- RabbitMQ 3.13.x
- JUnit 5
